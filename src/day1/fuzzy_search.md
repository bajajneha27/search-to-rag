
# Fuzzy Search & Tolerant Retrieval

What happens if the query has a typo?

Example:
- wirless mause

---

## Problem

Exact matching fails:
- "wirless" ≠ "wireless"

---

## Solution: Fuzzy Matching

- Edit distance  
- k-gram indexing  
- phonetic matching  
- dictionary-based correction  

---
