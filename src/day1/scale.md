# Search at Scale

Google does NOT search the web in real time. It searches a pre-built index.

---

## 3 Stages

### 1. Crawling
- discovers pages using bots ( Googlebot )
- follows links
- continuous process  

---

### 2. Indexing
- processes content  
- extracts signals  
- builds massive inverted indexes  
- stores multiple indexes (web, images, videos)

---

### 3. Serving
- processes query  
- retrieves candidates  
- ranks results  

---

## Ranking Signals

- relevance  
- quality  
- freshness  
- user context  

---

## Scale Challenges

- billions of documents  
- distributed systems  
- caching  
- parallel processing  
- constant updates
