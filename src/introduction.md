
# Introduction

This workshop explores how search systems work, from traditional keyword-based approaches to modern AI-powered systems.

---

### What you'll learn

- How traditional search works
- Why it fails in some cases
- How semantic search improves results
- What Retrieval-Augmented Generation (RAG) is

---

### Agenda

#### Session 1 — Foundations of Search (50 min)

- Introduction to Search Engine (5 min)
- Indexing (5 min)
- Inverted Index (10 min)
- Query Processing ( 10 min)
- Fuzzy Search (10 min)
- Limits of Traditional Search (5 min)
- Search at Scale (5 min)

#### Session 2 — Modern Search with AI (50 min)

- Introduction: Beyond Keywords (5 min)
- Transformer (5 min)
- Embeddings (10 min)
- Vector Databases (10 min)
- RAG ( Retrieval-Augmented Generation) (10 min)
