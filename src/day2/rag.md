
# 10. Retrieval-Augmented Generation

RAG combines retrieval with language models.

---

## Problem

LLMs:
- may hallucinate  
- lack real-time knowledge

---

## Solution

1. Retrieve relevant documents  
2. Pass them to LLM  
3. Generate answer

---

## Flow

Query → Retrieve → Generate

---

## Benefit

- grounded answers  
- more accurate  
- context-aware
