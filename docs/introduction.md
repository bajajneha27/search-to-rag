
# Introduction

This workshop explores how search systems work, from traditional keyword-based approaches to modern AI-powered systems.

We will build intuition through:
- demos
- simple mental models
- progressive improvements to search

---

## What you'll learn

- How traditional search works
- Why it fails in some cases
- How semantic search improves results
- What Retrieval-Augmented Generation (RAG) is

---

## Structure

Day 1: Traditional Search  
Day 2: AI-powered Search (RAG)
